# Sistema de Monitoreo de Equipos

Sistema de monitoreo en tiempo real para equipos en red.

## Tecnologías
- Node.js
- PostgreSQL
- Prisma
- Python (Agente)
- Chart.js

## Funcionalidades
- Monitoreo CPU/RAM
- Estado en tiempo real
- Dashboard web
- Historial de datos
